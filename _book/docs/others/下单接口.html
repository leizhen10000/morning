7.2 下单接口

7.2 渠道订单发单接口使用手册

公共约定

所有的请求返回格式统一定义为：

Key Value
Request header {" "}
Request params Context-type:form/data
Request body Content-type:application/json
Url params url参数拼接[?arg1=&arg2=]
Response body {
 "success":请求是否成功执行,true:成功;false:失败",
  "messageCode": "失败code编码，客户端可据此额外做业务逻辑",
  "message": "失败信息",
  "orderId": {"生成的订单号"}
}

渠道订单

1. 饿了么渠道订单（包含预约和非预约单）

Key Value
Path place/{environment}/channel/eleme
使用场景 下饿了么渠道订单
HTTP Method GET
Url params {
 "environment":"String 环境变量 qa/qa1" 必传
 "scheduled":"String 是否为预约单，0：非预约单，1，预约单"必传
 "timestamp":"String 预约时间，毫秒级别的时间戳"非必传
 如果非预约单，时间戳可以不用传；如果预约单，不传时间戳，默认半小时后
}
Response body {
 "success":请求是否成功执行,true:成功;false:失败",
  "messageCode": "失败code编码，客户端可据此额外做业务逻辑",
  "message": "失败信息",
  "orderId": {"生成的订单号"}
}

2. 普通渠道订单（包含预约和非预约单）

Key Value
Path place/{environment}/channel/normal
使用场景 下普通渠道订单，如 闪电购，花加等
HTTP Method GET
Url params {
 "environment":"String 环境变量 qa/qa1" 必传
 "scheduled":"String 是否为预约单，0：非预约单，1，预约单"必传
 "channelName":"String 渠道名称，名称必须和crm中名称一致" 包含 易果、闪电购、花加、安鲜达等必传
 "timestamp":"String 预约时间，毫秒级别的时间戳"
 如果非预约单，时间戳可以不用传；如果预约单，不传时间戳，默认半小时后
 "standardCategory":"String 标品类型 0：专送 1：快送2：蜂鸟商家版3：众包4：蜂鸟商家版-组织运力5：专快混合6：新零售99：其它"非必传
}
Response body {
 "success":请求是否成功执行,true:成功;false:失败",
  "messageCode": "失败code编码，客户端可据此额外做业务逻辑",
  "message": "失败信息",
  "orderId": {"生成的订单号"}
}

3. 菜鸟揽件兜底订单（包含预约单和非预约单）

Key Value
Path place/{environment}/lanjian
使用场景 下菜鸟揽件兜底订单
HTTP Method GET
Url params {
 "environment":"String 环境变量 qa/qa1" 必传
 "reservation":"String 是否为预约单，0：非预约单，1，预约单"必传
 "start":"String 时间戳毫秒 开始时间"非必传
 "end":"String 时间戳毫秒，结束时间"非必传但是start和end必须同时存在
 如果start和end两个时间都不传，默认30分钟start，40分钟end
}
Response body {
 "success":请求是否成功执行,true:成功;false:失败",
  "messageCode": "失败code编码，客户端可据此额外做业务逻辑",
  "message": "失败信息",
  "orderId": {"生成的订单号"}
}

6. 菜鸟揽件支付

Key Value
Path place/lanjian/pay
使用场景 菜鸟揽件支付
HTTP Method GET
Url params {
 "orderId":"String 订单id" 必传
 "riderId":"String 骑手id"必传 前提：骑手先在 app 上抢单
 "paid":"String 是否支付，1：已支付，0：未支付"非必传
 "fee""String 小费：如13.00"非必传
 "deliveryFee""String 配送费：如7.00"非必传
 "fee""String 小费：如13.00"非必传
 如果后面四个参数都没有传，系统会默认取上面的值
}
Response body {
 "success":请求是否成功执行,true:成功;false:失败",

接口使用范例

主要是按照下面模板调用接口，上面文档用于参考

  1. 饿了么预约单（带指定预约时间）
    (qa) http://192.168.11.39:46981/place/qa/channel/eleme?scheduled=1&timestamp=1501759800000 (qa1)http://192.168.11.39:46981/place/qa1/channel/eleme?scheduled=1&timestamp=1501759800000

  2. 饿了么即时单
    (qa) http://192.168.11.39:46981/place/qa/channel/eleme?scheduled=0 (qa1)http://192.168.11.39:46981/place/qa1/channel/eleme?scheduled=0

  3. 饿了么预约单（默认30分钟后预约）
    (qa) http://192.168.11.39:46981/place/qa/channel/eleme?scheduled=1 (qa1)http://192.168.11.39:46981/place/qa1/channel/eleme?scheduled=1

  4. 饿了么即时单（包含标品类型）
    (qa) http://192.168.11.39:46981/place/qa/channel/eleme?scheduled=0&standardCategory=1 (qa1)http://192.168.11.39:46981/place/qa1/channel/eleme?scheduled=0&standardCategory=1

  5. 普通渠道即时单
    (qa )http://192.168.11.39:46981/place/qa/channel/normal?scheduled=0&channelName=闪电购 (qa1)http://192.168.11.39:46981/place/qa1/channel/normal?scheduled=0&channelName=闪电购

  6. 普通渠道预约单（默认30分钟后预约） (qa )http://192.168.11.39:46981/place/qa/channel/normal?scheduled=1&channelName=闪电购 (qa1)http://192.168.11.39:46981/place/qa1/channel/normal?scheduled=1&channelName=闪电购

  7. 揽件兜底即时单
    (qa )http://192.168.11.39:46981/place/qa/lanjian?reservation=0 (qa1)http://192.168.11.39:46981/place/qa1/lanjian?reservation=0

  8. 揽件兜底预约单（默认 start 30分钟后，end 40分钟后）时间戳要自己传
    (qa) http://192.168.11.39:46981/place/qa/qalanjian?reservation=1 (qa 设置预约时间) http://192.168.11.39:46981/place/qa/lanjian?reservation=1&start=1502076600000&end=1502077200000 (qa1)http://192.168.11.39:46981/place/qa1/lanjian?reservation=1

  9. 揽件取件（骑手id来源，在app上抢单后该骑手id，记得修改订单id和骑手id）
    http://192.168.100.35:8081/place/lanjian/taken?orderId=215817&riderId=79

  10. 揽件妥投/完成订单
    http://192.168.100.35:8081/place/lanjian/finish?orderId=215817&riderId=79

  11. 揽件支付（完成前/完成后均可支付）
    http://192.168.100.35:8081/place/lanjian/pay?orderId=215817&riderId=79 http://192.168.100.35:8081/place/lanjian/pay?orderId=215817&riderId=79&paid=1&fee=13&insureFee=25&deliveryFee=7

  12. 同城 裹裹直送订单（qa环境）
    http://192.168.11.39:46981/place/qa/tongcheng

  13. 同城 裹裹直送订单（qa1环境） http://192.168.11.39:46981/place/qa1/tongcheng
